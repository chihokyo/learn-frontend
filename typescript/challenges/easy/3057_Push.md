# Push

## 1. 示例

在类型系统里实现通用的 `Array.push` 。

例如：

```typescript
type Result = Push<[1, 2], '3'>; // [1, 2, '3']
```

## 2. 自己实现

其实就是实现一个 push，在后面添加。

```typescript
type Push<T extends any[], U> = [...T, U];
```

## 3. 难点

### 难点 1 类型约束+拓展运算符

其实就是拓展运算符直接实现。没什么好说的。直接类型约束+拓展运算符就可以。

```typescript
type Push<T extends any[], U> = [...T, U];
```
